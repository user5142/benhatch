3 VS Code Shortcuts to Boost Your Dev Workflow

Ben Hatch

May 28, 2023

Visual Studio Code is a free source-code editor developed by Microsoft. It includes support for debugging, embedded Git control, syntax highlighting, intelligent code completion and more.

I’ve been using VS Code since December of 2022 when I kicked off my web development journey and I absolutely love working with it. It’s nicely designed, fast and has a great UX.

Through my heavy use of VS Code while chipping away at Codecademy’s Full-Stack Engineer Path over the past 6 months, I’ve picked up some handy tricks to speed up my coding process.

Read on for 3 VS Code shortcuts I use every day to boost my development workflow.

Ctrl + `

This shortcut opens and closes the integrated terminal in VS Code. No need to switch back and forth between your code editor and terminal window!

This shortcut is very helpful when you need to change directories, run a program or fire off some git commands.

Shift + Alt + F

This shortcut automatically formats your code to make it look nice and neat and follow best formatting practices.

This shortcut works out of the box with HTML, CSS, JavaScript and a number of other languages. To get it to work for other formats such as markdown, you’ll need to install special formatters.

Alt + L + O

You must have the awesome (and free) Live Server plugin installed for this shortcut, which launches a live instance of your web project in the browser.

This shortcut is amazing for getting continuous visual feedback and running manual tests on a web project as you build it out.

I hope these simple VS Code shortcuts can help give you a nice workflow boost and inch you closer to becoming a 10x dev. Happy coding!
